🔥 FIREREACH - QUICK START GUIDE
================================

STEP 1: GET API KEYS (5 minutes)
---------------------------------
1. Groq: https://console.groq.com → Create account → Get API key
2. SerpAPI: https://serpapi.com → Sign up → Get API key (100 free searches/month)
3. NewsAPI: https://newsapi.org → Register → Get API key (1000 free requests/month)
4. Gmail App Password: 
   - Go to https://myaccount.google.com/apppasswords
   - Generate app password for "Mail"
   - Copy the 16-character password


STEP 2: SETUP BACKEND (2 minutes)
----------------------------------
cd firereach/backend
pip install -r requirements.txt
cp .env.example .env

Edit .env file and add your keys:
GROQ_API_KEY=gsk_your_actual_key
SERP_API_KEY=your_actual_key
NEWS_API_KEY=your_actual_key
SMTP_USER=your-gmail@gmail.com
SMTP_APP_PASSWORD=xxxx xxxx xxxx xxxx

Start backend:
uvicorn main:app --reload

Backend will run on: http://localhost:8000


STEP 3: SETUP FRONTEND (2 minutes)
-----------------------------------
cd firereach/frontend
npm install
cp .env.local.example .env.local

Edit .env.local:
NEXT_PUBLIC_API_URL=http://localhost:8000

Start frontend:
npm run dev

Frontend will run on: http://localhost:3000


STEP 4: TEST IT (1 minute)
---------------------------
1. Open http://localhost:3000
2. Fill the form:
   - ICP: "We sell high-end cybersecurity training to Series B startups."
   - Target Company: "Deel"
   - Email: your-email@gmail.com
   - Name: Alex
3. Click "Launch Agent 🚀"
4. Watch the magic happen!


TROUBLESHOOTING
---------------
Backend won't start?
→ Check Python version: python --version (need 3.11+)
→ Check if .env file has all 4 keys

Frontend won't start?
→ Check Node version: node --version (need 18+)
→ Run: rm -rf node_modules && npm install

No signals found?
→ Check SerpAPI key is valid
→ Try a well-known company like "Stripe" or "Deel"

Email not sending?
→ Check Gmail SMTP credentials
→ Make sure you're using App Password (not regular password)
→ Enable "Less secure app access" if needed
