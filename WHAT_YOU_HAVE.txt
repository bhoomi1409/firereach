WHAT YOU NEED TO DO:
--------------------

1. GET 4 API KEYS (5 minutes):
   - Groq: https://console.groq.com
   - SerpAPI: https://serpapi.com
   - NewsAPI: https://newsapi.org
   - Resend: https://resend.com

2. ADD KEYS TO backend/.env:
   GROQ_API_KEY=gsk_...
   SERP_API_KEY=...
   NEWS_API_KEY=...
   RESEND_API_KEY=re_...

3. RUN SETUP:
   cd firereach
   ./SETUP_COMMANDS.sh

4. START BACKEND:
   cd backend
   uvicorn main:app --reload

5. START FRONTEND (new terminal):
   cd frontend
   npm run dev

6. TEST IT:
   Open http://localhost:3000
   Fill form with Rabbitt Challenge data
   Click "Launch Agent 🚀"
   Check your email!
